New in version 1.3.0
================================

|:sparkles:| Features
-----------------------------------------------------------
* Multithreading supported for all functions. (see :doc:`/api/multithreading_asynchronous`)
* Asynchronous functions supported for all functions. (see :doc:`/api/multithreading_asynchronous`)

.. note::

    The v1.3.0 also dropped the support of the mutli-versioned documentation, which means the documentation is now only available for the latest version of PyNiryo.

    If you need the documentation for an older version, you can build it locally by
    checking out the corresponding git tag and running:

    .. code-block:: bash

        pip install -r docs/requirements.txt
        sphinx-build docs/ _build/html
